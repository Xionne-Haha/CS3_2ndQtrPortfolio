Content: A comprehensive, interactive listing of every genre covered (e.g., Action RPG, Real-Time Strategy, First-Person Shooter). C. Deep Dives (Analysis)

Purpose: To provide the in-depth, comparative analysis of the two defining games for each genre.

Content: This page is the destination for the site's core academic content. It begins with the full historical context and evolutionary timeline of the selected genre.
